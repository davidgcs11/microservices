# Microservices
A simple application to test microservices, will be using 1 client (ApiGateway), 1 Microservice and Redis 🥳

# How to run
1. Just clone the repo and run
```bash
$ docker-compose up
```
2. Once completed try doing a POST request
```bash
localhost:3000
```
